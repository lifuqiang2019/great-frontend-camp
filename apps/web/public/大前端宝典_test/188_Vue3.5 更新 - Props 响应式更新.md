# Vue3.5 更新 - Props 响应式更新

- Vue3 需要配合使用 toRefs 或者 toRef 解构，才能保持响应式。

```javascript
// index.vue  
<input v-model="msg" style="width: 300px;"/>  
<child :msg="msg" ref="childRef"/>  
  
import { ref } from 'vue';  
const msg = ref('');

```

```javascript
// child.vue  
<div class="green">{{ msg }}</div>  
  
import { watch, toRef } from 'vue';  
const props = defineProps(\[ 'msg' \]);  
const msg = toRef(props, 'msg');  
watch(msg, () => {  
console.log('watch msg:', msg.value);  
});

```
- Vue3.5 直接进行解构赋值，响应式会保留。

```javascript
// index.vue  
<input v-model="msg" style="width: 300px;"/>  
<child :msg="msg" />  
  
import { ref } from 'vue';  
const msg = ref('');

```

```javascript
// child.vue  
<div class="green">{{ msg }}</div>  
  
import { watch } from 'vue';  
const { msg } = defineProps(\[ 'msg' \]);  
watch(() => msg, () => {  
console.log('watch msg:', msg);  
});

```
