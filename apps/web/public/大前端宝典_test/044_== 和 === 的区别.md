# == 和 === 的区别

- "==" ，先隐式类型转换，再判断值是否相等
- "==="，直接判断 类型 + 值 是否相等

```javascript
console.log(1 == "1"); // true，因为在比较之前，字符串"1"会被转换为数值1  
console.log(true == 1); // true，因为在比较之前，布尔值true会被转换为数值1

```

```javascript
console.log(1 === "1"); // false，因为它们的类型不同  
console.log(true === 1); // false，因为它们的类型不同
```
