# Vue3.5 更新 - useTemplateRef

- vue3 需要通过定义一个 ref(null) 引用一个组件，但这样对语义化层面理解不太友好。

```javascript
// index.vue  
<child ref="childRef"/>  
<button @click="handler()">getMsg</button>  
  
import { ref } from 'vue';  
import child from './components/child.vue'  
const childRef = ref(null);  
function handler() {  
console.log('msg:', childRef.value.getMsg());  
}

```

```javascript
// child.vue  
function getMsg() {  
return 'i am message';  
}  
  
defineExpose({  
getMsg  
});

```
- vue3.5 新增了 useTemplateRef 方法作为模板引用，语义化更好。

```javascript
// index.vue  
<child ref="childRef"/>  
<button @click="handler()">getMsg</button>  
  
import { useTemplateRef } from 'vue';  
import child from './components/child.vue'  
const childRef = useTemplateRef('childRef');  
function handler() {  
console.log('msg:', childRef.value.getMsg());  
}

```

```javascript
// child.vue  
function getMsg() {  
return 'i am message';  
}  
  
defineExpose({  
getMsg  
});

```
