# 实现 b 函数，使得 b.then 中能打印出 cb 的参数值

**补充说明：即打印 Math.random() 的值**

```javascript
const fn = (cb) => {  
setTimeout(() => {  
cb(Math.random());  
}, 1000);  
}  
  
const b = () => {  
// 实现代码  
}  
  
b().then(res => {  
console.log(res);  
})

```
**实现：利用 promise resolve**

```javascript
const fn = (cb) => {  
setTimeout(() => {  
cb(Math.random());  
}, 1000);  
}  
  
const b = () => {  
return new Promise(resolve => {  
fn(resolve);  
});  
}  
  
b().then(res => {  
console.log(res);  
})

```
