# 强制类型转换、隐式类型转换

- 强制类型转换

```javascript
var num = Number("42"); // 强制将字符串转换为数字  
var str = String(123); // 强制将数字转换为字符串  
var bool = Boolean(0); // 强制将数字转换为布尔值

```
- **隐式类型转换**

```javascript
var result = 10 + "5"; // 隐式将数字和字符串相加，结果为字符串 "105"  
true == 1; // 隐式将布尔值转换为数字 1  
false == 0; // 隐式将布尔值转换为数字 0

```
