# 链式调用实现方式

链式调用是通过在对象的方法中返回对象自身（this）来实现的。可使多个方法调用连续写在一起，形成链式调用。

```javascript
class Calculator {  
constructor(num) {  
this.value = num;  
}  
  
add(num) {  
this.value += num;  
return this; // 返回自身，以实现链式调用  
}  
  
subtract(num) {  
this.value -= num;  
return this;  
}  
  
multiply(num) {  
this.value \*= num;  
return this;  
}  
  
divide(num) {  
this.value /= num;  
return this;  
}  
  
getValue() {  
return this.value;  
}  
}  
  
const calculator = new Calculator(10);  
const result = calculator.add(5).subtract(2).multiply(3).divide(4).getValue();  
  
console.log(result); // 输出 9.75

```
