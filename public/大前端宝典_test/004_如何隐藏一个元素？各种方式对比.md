# 如何隐藏一个元素？各种方式对比

在 CSS 中隐藏元素有多种方式，它们的行为（是否占据空间、是否响应事件、是否回流）各不相同。

| 方式 | 语法 | 占据空间? | 响应点击? | 触发回流? | 说明 |
| :--- | :--- | :---: | :---: | :---: | :--- |
| **完全移除** | `display: none` | ❌ 否 | ❌ 否 | ✅ 是 | 元素从渲染树中移除，如同不存在。 |
| **视觉隐藏** | `visibility: hidden` | ✅ 是 | ❌ 否 | ❌ 否 | 元素不可见，但保留占位，只触发重绘。 |
| **透明度** | `opacity: 0` | ✅ 是 | ✅ 是 | ❌ 否 | 元素完全透明，但仍然可以点击交互。 |
| **位置移出** | `position: absolute; left: -9999px;` | ❌ 否 | ❌ 否 | ✅ 是 | 将元素移出可视区域，通常用于无障碍阅读。 |
| **裁剪** | `clip-path: circle(0)` | ✅ 是 | ❌ 否 | ❌ 否 | 将元素裁剪为 0 大小，保留原始占位。 |

## 总结
*   如果希望元素**彻底消失**（不占位），使用 `display: none`。
*   如果希望元素**不可见但保留位置**（如为了布局稳定），使用 `visibility: hidden`。
*   如果希望元素**不可见但仍可点击**（如透明按钮），使用 `opacity: 0`。
