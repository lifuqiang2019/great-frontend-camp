# CSS overflow 属性详解

`overflow` 属性规定当内容溢出元素框（Content Box）时发生的事情。

## 属性值对比

### 1. `visible` (默认值)
*   **表现**：内容不会被修剪，会呈现在元素框之外。
*   **后果**：溢出的内容可能会覆盖页面上的其他元素。

### 2. `hidden`
*   **表现**：溢出的内容会被修剪（隐藏），其余内容不可见。
*   **后果**：用户无法看到被隐藏的内容（没有滚动条）。这也是触发 BFC 的常用方式之一。

### 3. `scroll`
*   **表现**：无论内容是否溢出，浏览器都会显示滚动条。
*   **特点**：如果内容未溢出，滚动条处于不可用状态（灰色），但依然占据空间。

### 4. `auto`
*   **表现**：浏览器自动处理。
*   **特点**：如果内容溢出，则显示滚动条；如果未溢出，则不显示。这是最常用的设置。

### 5. `inherit`
*   **表现**：从父元素继承 `overflow` 属性的值。
