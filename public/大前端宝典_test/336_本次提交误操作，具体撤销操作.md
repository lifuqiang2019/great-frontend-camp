# 本次提交误操作，具体撤销操作

1.  **撤销上一个提交：**

```bash
git reset --soft HEAD^

这会将 HEAD（当前分支的最新提交）移动到前一个提交，并将上一个提交从历史中移除。

**注意：** 需要小心使用此命令，因为它会影响整个项目历史。

1.  **撤销多个提交：**

```

```bash
git reset HEAD~n # 将 HEAD 指向要保留的提交之前的提交（n 为要保留的提交数）  
git stash # 暂存要丢弃的提交

1.  **回滚提交：**

```

```bash
git revert <commit-hash>

<commit-hash> 是要回滚的提交的哈希值。

```
