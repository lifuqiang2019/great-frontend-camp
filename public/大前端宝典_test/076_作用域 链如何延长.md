# 作用域 链如何延长

- **闭包**

```javascript
function makeCounter() {  
var count = 0;  
return function() {  
count++;  
return count;  
};  
}  
  
var counter1 = makeCounter();  
var counter2 = makeCounter();  
  
console.log(counter1()); // 1  
console.log(counter1()); // 2  
  
console.log(counter2()); // 1，每个 counter 具有自己的作用域链，且都延长了 count 的作用域

```
