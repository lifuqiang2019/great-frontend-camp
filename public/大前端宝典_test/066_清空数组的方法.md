# 清空数组的方法

```javascript
// 清空前  
let list = \[1,2,3,4,5\];  
  
// 逻辑实现 ???  
  
// 清空后  
console.log(list); // \[\]

```
- **方法一**

```javascript
list = \[\]  
// 复杂度：O(1). 只是重新绑定一个新数组引用，不涉及旧数组的逐一删除。  
  
// 缺点：如果有其它变量引用了旧数组，不会被清空。  
let arr = \[1, 2, 3\];  
let ref = arr;  
arr = \[\];  
console.log(ref); // \[1, 2, 3\] ✅ ref 还是旧内容

```
- **方法二**

```javascript
arr.splice(0, arr.length)  
// 复杂度：O(n)（n 为数组长度）。因为 splice 需要返回被删除的元素，内部会对每个元素进行操作。

```
- **方法三**

```javascript
while (arr.length) {  
arr.pop();  
}  
// 复杂度：pop() 每次删除数组最后一个元素，复杂度是 O(1)，但需要执行 n 次，所以复杂度是 O(n)

```
- **方法四**

```javascript
while (arr.length) {  
arr.shift();  
}  
// 复杂度：O(n²) shift() 删除数组第一个元素时，需要将剩余的所有元素整体前移。

```
- **方法五**

```javascript
arr.length = 0;  
// **优点**：原地清空，所有引用同步清空。最彻底最高效的清空

```
