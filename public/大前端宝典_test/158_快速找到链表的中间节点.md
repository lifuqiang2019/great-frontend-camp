# 快速找到链表的中间节点

快慢指针，快指针步进为2，慢指针步进为1，两个指针同时启动，当快指针走到底，慢指针指向的即是中间节点。

```javascript
class ListNode {  
constructor(val, next = null) {  
this.val = val;  
this.next = next;  
}  
}  
  
function findMiddleNode(head) {  
let slow = head;  
let fast = head;  
while (fast !== null && fast.next !== null) {  
slow = slow.next;  
fast = fast.next.next;  
}  
return slow;  
}

```
